Wiki + Gists = Wigiki

The concept behind wigiki was to create a tool which can group and organize all my random notes that I write from time to time as Gists.

You supply a json configuration file and a folder where some template files reside and it generates a static html site. The site can be deployed on your own server or on Github (using the static pages feature).
